l9gpu Helm chart repository

Vendor-agnostic GPU telemetry for AI/ML and HPC clusters.

Add the repo

helm repo add l9gpu https://last9.github.io/gpu-telemetry
helm repo update
helm search repo l9gpu

Install the chart

helm install l9gpu l9gpu/l9gpu \
  --namespace l9gpu --create-namespace \
  --set otlp.endpoint=$OTEL_EXPORTER_OTLP_ENDPOINT
